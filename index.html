Flood Fill Algorithm

This sketch is a visualization of a flood fill algorithm using the HTML5 imageData object for pixel manipulation. A flood fill algorithm is used to determine the properties of the area around a particular cell in a block. In this case, the algorithm will be mimicking the bucket fill tool of the Microsoft Paint program. There are also options to draw freely around and insert random, colorful, and different size rectangles.
